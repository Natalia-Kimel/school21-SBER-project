# СберАссистент — корпоративный ИИ-сервис

Единая точка доступа к базе знаний, персональному онбордингу, суммаризации встреч, анализу PDF/DOCX/XLSX/CSV/TXT/MD и личным задачам.

## Функционал

- вход под шестью ролями и ролевое меню;
- поиск по корпоративной базе знаний с разграничением доступа;
- RAG-ассистент: извлекает релевантные фрагменты, формирует ответ, показывает источники, уверенность и рекомендуемые документы;
- подключение реальной генеративной модели через корпоративный OpenAI-compatible шлюз;
- загрузка и индексация документов экспертами и администраторами;
- персональный онбординг нового сотрудника;
- суммаризация встреч, выделение решений, рисков и создание задач;
- анализ файлов с краткой сводкой, ключевыми темами, числами и связанными документами;
- канбан задач;
- аналитика качества и журнал аудита;
- адаптивный интерфейс для компьютера и мобильного экрана.

## Запуск

### Windows

```bat
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
run.bat
```

### macOS / Linux

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
./run.sh
```

Откройте `http://127.0.0.1:8000`.

### Docker

```bash
cp .env.example .env
docker compose up --build
```

## Генеративная модель

Автономный режим `LLM_PROVIDER=local` работает без ключей и формирует проверяемый ответ из найденных фрагментов. Для полноценной генерации создайте `.env` на основе `.env.example` и укажите корпоративный OpenAI-compatible endpoint:

```env
LLM_PROVIDER=openai_compatible
LLM_API_URL=здесь должен быть корпоративный OpenAI-compatible endpoint
LLM_API_KEY=...
LLM_MODEL=...
```

Контекст RAG передаётся модели только после фильтрации по роли пользователя. При ошибке шлюза сервис автоматически возвращается в локальный безопасный режим.

## Структура

- `app/main.py` — API и маршруты;
- `app/db.py` — SQLite, RBAC, FTS5-индекс, задачи, онбординг, аудит;
- `app/ai.py` — RAG, локальный синтез и адаптер генеративной модели;
- `app/parsers.py` — извлечение текста из файлов;
- `static/` — готовый SPA-интерфейс;
- `seed_documents/seed.json` — демонстрационная база знаний;
- `data/sber_assistant.db` — создаётся автоматически.
